# SUTPC 大屏图层字段矩阵

生成日期：2026-06-21

范围：基于已有 SUTPC 大屏地图交互摘要、GIS 数据需求详表、BIM 集成摘要和本地 `E:\Data` 资产索引，整理大屏图层分组、核心字段和本地样例。本文不读取、移动、删除、上传原始资料。

## 结论

大屏图层应按“业务场景”而不是按文件格式组织。推荐分为六组：底图、项目、设施、BIM/三维、监测视频、周边专题。每组都需要明确可看、可查、可分析和联动对象。

## 最小接口字段模板

### 项目图层

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `project_id` | 是 | 项目唯一编号 |
| `name` | 是 | 项目名称 |
| `stage` | 是 | 规划、前期、建设、竣工、养护等阶段 |
| `geometry` | 是 | 点、线、面或范围 |
| `contract_section` | 可选 | 合同段信息 |
| `progress` | 可选 | 进度百分比或状态 |
| `investment` | 可选 | 投资额、累计产值等 |
| `model_refs` | 可选 | BIM/geokey/model_id 列表 |
| `video_refs` | 可选 | 现场视频列表 |

### 设施图层

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `asset_id` | 是 | 设施唯一编码 |
| `asset_type` | 是 | 桥梁、隧道、边坡、场站等 |
| `name` | 是 | 设施名称 |
| `road` | 可选 | 所属道路 |
| `geometry` | 是 | 空间位置 |
| `condition_level` | 可选 | 技术状况等级 |
| `maintenance_refs` | 可选 | 养护项目 |
| `monitor_refs` | 可选 | 监测点 |
| `model_refs` | 可选 | BIM/三维模型 |

### BIM 模型图层

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `model_id` | 是 | 模型唯一编号 |
| `geokey` | 是 | BIM 支撑平台/T-GIS 加载标识 |
| `url` | 是 | 模型服务地址 |
| `stage` | 是 | 初设、施工图、施工、竣工、运维 |
| `format` | 是 | 3D Tiles、S3M、B3DM、glTF、FBX 等 |
| `project_id` | 可选 | 关联项目 |
| `asset_id` | 可选 | 关联设施 |
| `version` | 可选 | 版本或更新时间 |

### 视频/监测图层

| 字段 | 必填 | 说明 |
| --- | --- | --- |
| `monitor_id` / `video_id` | 是 | 点位或视频编号 |
| `type` | 是 | 监测类型或视频类型 |
| `geometry` | 是 | 坐标位置 |
| `status` | 是 | 在线、离线、预警、正常等状态 |
| `warning_level` | 可选 | 预警等级 |
| `stream_url` | 可选 | 视频流地址，文档中不保存敏感真实地址 |
| `project_id` | 可选 | 关联项目 |
| `asset_id` | 可选 | 关联设施 |

## 场景模式建议

| 场景模式 | 默认打开图层 | 默认关闭图层 | 目的 |
| --- | --- | --- | --- |
| 全市总览 | 底图、项目、重点设施、预警 | BIM 细节、全部视频、POI 全量 | 控制信息密度 |
| 建设监管 | 在建项目、合同段、视频、监测点、BIM | 学校、医院等周边 POI | 聚焦建设过程 |
| 设施管养 | 桥梁、隧道、边坡、技术状况、养护项目 | 规划项目、施工过程图层 | 聚焦设施状态 |
| BIM 模型 | BIM/三维、构件属性、项目进度 | POI、交通运行全量 | 聚焦模型和构件 |
| 应急/预警 | 预警点、视频、医院、政府单位、路网 | BIM 细节、历史资料 | 支持处置和研判 |

## 当前缺口

| 缺口 | 影响 | 后续动作 |
| --- | --- | --- |
| 缺少真实服务 URL/geokey | 无法直接形成可加载模型图层 | 后续建立模型发布字段模板 |
| 坐标系未统一核查 | BIM、3D Tiles、矢量图层可能无法叠合 | 建立坐标系检查清单 |
| 视频流地址敏感 | 不适合写入公开知识库 | 只保存元数据，不保存真实流地址 |
| 本地样例和业务字段未完全对应 | 资产卡片和图层字典还需互链 | 后续更新模型资产卡片和数据资产索引 |
