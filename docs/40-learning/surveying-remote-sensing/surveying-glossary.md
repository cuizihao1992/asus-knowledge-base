# 测绘与遥感术语表

生成日期：2026-06-20

范围：基于 `docs/generated/surveying-satellite-manuals.md`、`docs/summaries/remote-sensing-image-stitching-papers.md`、`docs/05-data-assets-index.md` 中已索引的资料，整理常见测绘、遥感、GIS/BIM 和三维数据术语。本文是知识库工作解释，不替代正式国家标准、行业标准或产品手册。

## 使用方式

这个术语表用于读本地资料时快速建立上下文：

- 读测绘标准时，先看 DEM、DOM、DLG、DRG、比例尺、质量评定。
- 读遥感论文时，先看 RPC、RFM、正射校正、传感器模型、几何定标。
- 读三维 GIS/BIM 数据时，先看 3D Tiles、B3DM、CMPT、glTF、FBX、Shapefile。
- 整理项目资料时，用“对象、格式、坐标、精度、成果类型、关联文档”来描述资产。

## 测绘成果类型

| 术语 | 中文含义 | 工作解释 | 本地资料关联 |
| --- | --- | --- | --- |
| DEM | 数字高程模型 | 用规则格网或其他结构表达地表高程。常用于地形分析、坡度坡向、三维地形、正射校正。 | `CHT 1015.2`、`CHT 9009.2`、`CHT_9008.2` |
| DOM | 数字正射影像图 | 经过几何纠正和正射校正的影像成果，具有地图量测属性。 | `CHT 1015.3`、`CHT 9009.2`、`CHT-9008.3` |
| DLG | 数字线划图 | 以点、线、面等矢量形式表达地物、地貌和地理要素。 | `CHT 1011`、`CHT 1015.1` |
| DRG | 数字栅格地图 | 纸质地图或地图成果经扫描、纠正后形成的栅格地图。 | `CHT 1015.4`、`CHT 9009.2` |
| 数字测绘成果 | 测绘成果的数字化表达 | 包括 DEM、DOM、DLG、DRG、数字地形图等成果。 | `GBT 18316`、`CHT 1017` |
| 基础地理信息 | 基础空间地理数据 | 支撑 GIS、规划、交通、城市管理等业务的底层地理信息。 | `GBT 20258.3` |
| 地形图 | 表达地物、地貌、高程和地图要素的地图 | 可按 1:500、1:1000、1:2000、1:25000、1:50000 等比例尺组织。 | `GBT 12343.1`、`GBT 15661` |

## 航测和影像生产

| 术语 | 中文含义 | 工作解释 | 本地资料关联 |
| --- | --- | --- | --- |
| 航测 | 航空摄影测量 | 通过航空影像获取地形、地物和空间信息。 | `GBT 12340`、`GBT 12341`、`GBT 15661` |
| 航测内业 | 航空摄影测量内业 | 在室内完成像片处理、空三加密、测图、编辑、检查等工作。 | `GBT 12340`、`GBT 7930` |
| 航测外业 | 航空摄影测量外业 | 现场控制测量、像控点布设、调绘、补测等外业工作。 | `GBT 12341` |
| 像控点 | 影像控制点 | 在影像上可识别、地面坐标已知的控制点，用于几何校正和空三。 | 遥感几何论文、航测规范 |
| 空三 | 空中三角测量 | 利用影像连接点和控制点解算摄影测量几何关系。 | 航测规范、图像拼接资料 |
| 正射校正 | Orthorectification | 利用传感器模型、姿轨数据、DEM 等消除地形起伏和成像几何造成的变形。 | RPC/RFM 论文、DOM 标准 |
| 几何纠正 | Geometric Correction | 对影像几何畸变进行校正，使其符合地图坐标或参考影像。 | 遥感几何论文 |
| 辐射校正 | Radiometric Correction | 对影像灰度/辐射值进行校正，使不同时间、传感器或条件下更可比。 | 卫星产品手册 |
| 影像配准 | Image Registration | 将多幅影像对齐到同一坐标或像素空间。 | 相位相关、SIFT/RANSAC 资料 |
| 图像拼接 | Image Stitching | 将多幅有重叠区域的影像融合成更大范围影像。 | `图像拼接` 资料组 |

## 比例尺、分辨率和精度

| 术语 | 中文含义 | 工作解释 | 本地资料关联 |
| --- | --- | --- | --- |
| 比例尺 | 地图距离和实地距离的比例 | 比例尺越大，表示越详细；1:500 比 1:50000 更详细。 | `分辨率与比例尺.pdf` |
| 地面分辨率 | Ground Resolution | 一个像素对应的地面实际尺寸，常用米/像素表示。 | 卫星手册、遥感资料 |
| 空间分辨率 | Spatial Resolution | 传感器区分地物空间细节的能力，常和地面分辨率相关。 | 卫星产品手册 |
| 高程精度 | Elevation Accuracy | DEM 或高程成果与真实高程的误差水平。 | DEM 标准、`分辨率与比例尺6添加高程精度影响因素.doc` |
| 平面精度 | Horizontal Accuracy | 点位平面坐标的误差水平。 | 测绘质量评定资料 |
| 质量评定 | Quality Assessment | 对测绘成果完整性、位置精度、属性精度、逻辑一致性等进行检查和评价。 | `CHT 1017`、`GBT 18316` |
| 质量检查 | Quality Inspection | 在成果交付前按规范检查数据质量。 | `GBT 18316` |

## 遥感传感器和几何模型

| 术语 | 中文含义 | 工作解释 | 本地资料关联 |
| --- | --- | --- | --- |
| Pushbroom | 推扫式成像 | 线阵传感器随平台运动逐行获取影像，常见于高分辨率光学卫星。 | Pushbroom 卫星几何资料 |
| CCD | 电荷耦合器件 | 常见影像传感器类型。线阵 CCD 几何误差会影响影像定位精度。 | CCD 畸变定标资料 |
| 线阵相机 | Linear Array Camera | 一次成像一行或多行，依靠平台运动形成二维影像。 | Pushbroom、CCD 资料 |
| 严格传感器模型 | Rigorous Sensor Model | 基于物理成像几何、姿轨数据、相机参数建立的传感器模型。 | 黄师兄论文、RFM 资料 |
| RPC | Rational Polynomial Coefficients | 有理多项式系数，用多项式近似传感器成像几何，常用于卫星影像定位。 | `基于RPC模型的IRS_P5影像正射校正` |
| RFM | Rational Function Model | 有理函数模型，和 RPC 相关，用于描述像方和物方坐标关系。 | 黄师兄论文 |
| 几何定标 | Geometric Calibration | 估计或修正传感器几何参数、畸变、安装误差等。 | Pushbroom 交叉定标资料 |
| 交叉定标 | Cross Calibration | 利用不同传感器、不同影像或参考数据进行相对几何/辐射校准。 | `Geometric_Cross_Calibration...` |
| 镜头畸变 | Lens Distortion | 镜头成像导致的位置偏差，包括径向、切向、偏心畸变等。 | `Decentering Distortion of Lenses.pdf` |
| 视角计算 | Viewing Angle Calculation | 计算传感器到地面目标的观测角度，影响几何定位和辐射分析。 | 黄师兄论文 |

## 图像匹配和配准算法

| 术语 | 中文含义 | 工作解释 | 本地资料关联 |
| --- | --- | --- | --- |
| SIFT | 尺度不变特征变换 | 提取尺度和旋转相对稳定的局部特征，用于匹配和拼接。 | 图像拼接资料 |
| SURF | 加速稳健特征 | 类似 SIFT 的局部特征算法，速度更快。 | 图像拼接论文 |
| RANSAC | 随机采样一致性 | 在存在误匹配时估计稳健模型，如单应矩阵、基础矩阵。 | `RUNSAC` 资料 |
| Homography | 单应矩阵 | 描述两个平面视图之间的投影变换，常用于图像拼接。 | `mosaic` MATLAB 代码 |
| 相位相关 | Phase Correlation | 基于频域相位估计图像平移，常用于亚像素配准。 | 相位相关论文 |
| 亚像素配准 | Subpixel Registration | 精度小于一个像素的影像对齐。 | `A Fast Direct Fourier-Based...` |
| 光流 | Optical Flow | 表示图像序列中像素运动的矢量场。 | 光流评估、视频目标检测资料 |
| 特征匹配 | Feature Matching | 在多幅影像中找到相同地物或纹理点。 | 图像拼接资料 |
| 影像融合 | Image Blending | 拼接后对接缝、亮度、颜色进行平滑融合。 | 图像拼接资料 |
