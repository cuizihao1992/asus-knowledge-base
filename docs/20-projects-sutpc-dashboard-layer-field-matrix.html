最小接口字段模板

项目图层

字段必填说明
project_id是项目唯一编号
name是项目名称
stage是规划、前期、建设、竣工、养护等阶段
geometry是点、线、面或范围
contract_section可选合同段信息
progress可选进度百分比或状态
investment可选投资额、累计产值等
model_refs可选BIM/geokey/model_id 列表
video_refs可选现场视频列表

设施图层

字段必填说明
asset_id是设施唯一编码
asset_type是桥梁、隧道、边坡、场站等
name是设施名称
road可选所属道路
geometry是空间位置
condition_level可选技术状况等级
maintenance_refs可选养护项目
monitor_refs可选监测点
model_refs可选BIM/三维模型

BIM 模型图层

字段必填说明
model_id是模型唯一编号
geokey是BIM 支撑平台/T-GIS 加载标识
url是模型服务地址
stage是初设、施工图、施工、竣工、运维
format是3D Tiles、S3M、B3DM、glTF、FBX 等
project_id可选关联项目
asset_id可选关联设施
version可选版本或更新时间

视频/监测图层

字段必填说明
monitor_id / video_id是点位或视频编号
type是监测类型或视频类型
geometry是坐标位置
status是在线、离线、预警、正常等状态
warning_level可选预警等级
stream_url可选视频流地址，文档中不保存敏感真实地址
project_id可选关联项目
asset_id可选关联设施
