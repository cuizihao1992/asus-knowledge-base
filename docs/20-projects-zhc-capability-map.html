3. 闭环优先

来源：mini-transformer 和 quant-system

一个项目是否值得优先整理，优先看它是否有闭环：

输入 -> 处理 -> 结果 -> 保存/展示 -> 下一步动作

当前例子：

  • mini-transformer：文本 -> 训练 -> checkpoint -> 生成 -> API 服务。
  • quant-system：行情 -> SQLite -> 回测 -> 结果 -> 信号监控。
  • 当前知识库：本地资料 -> 索引 -> Markdown -> HTML -> Pages 发布。
