目标

建立一条可重复的模型发布验证链路：

FBX 原始模型
  -> 临时工作区副本
  -> Blender / FBX SDK / assimp 打开验证
  -> glTF / GLB 导出
  -> Cesium 或 Three.js 预览
  -> 必要时转 3D Tiles
  -> 记录坐标、比例、材质、贴图和业务属性损失
