目标：把本地资料先按模块建立索引，再逐步把高价值资料整理成摘要、随笔和专题文档。

- `E:\学习资料`

## 模块 6：待整理

来源目录：

- `E:\待整理文档`
- `E:\Download`
- `E:\360Downloads`
- `E:\迅雷下载`
- `D:\360Downloads`
- `D:\tmp`

收录内容：

- 暂时无法判断归属的下载文件。
- 需要筛选的压缩包。
- 重复或过期资料的清理记录。

建议标签：

- `待整理`
- `待筛选`
- `下载`
- `压缩包`
- `可能重复`
- `可删除候选`

处理规则：

- 先记录路径，不急着移动文件。
- 能归类的转入正式模块。
- 无价值或重复文件记录为“可删除候选”。
- 大压缩包先记录来源和可能内容，再决定是否解压。

## 模块 7：工具与安装包索引

来源目录：

- `E:\软件安装包`
- `E:\工具软件`
- `D:\Program Files`
- `D:\Programs`
- `D:\apache-tomcat`
- `D:\platform-tools`

收录内容：

- 工具名称。
- 版本。
- 用途。
- 安装路径。
- 是否仍在使用。

建议标签：

- `工具`
- `安装包`
- `环境`
- `SDK`
- `服务`
- `可删除候选`

处理规则：

- 不把二进制文件复制进知识库。
- 只记录索引和用途。
- 对仍在使用的环境记录启动方式和配置位置。
